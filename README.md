# Angel's Wink

Angel's Wink is a Vite + React marketing site for Angel Zino, positioning the brand as a tattoo-led beauty studio with training, digital growth support, and wellness services.

## Tech Stack

- React
- Vite
- GSAP
- Lenis
- React Router

## Local Development

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 3001
```

## Production Build

```bash
npm run build
```
